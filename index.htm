FitGem

A simple Fitbit CLI Client

Are you here to link your Fitbit account to FitGem? Click Here for instructions.

Easy to use

In FitGem, to access your step count, just use this command:

      

        
          
            $ fitgem steps

It's that easy!

Secure access

FitGem uses the latest OAuth 2.0 standards to keep you safe from data attacks.

Very fast

FitGem is built upon Ruby 2.3 and the latest httparty and multi-json libraries.

Installation

FitGem can be installed in the RubyGems format on Linux and Mac OS X alike. 

      

        
          
            $ sudo gem install fitgem-app

Note: sudo is used on Mac OS X and Debian-like systems to grant you super powers (root user access) and may be different on certain systems.

Authorization

To set up FitGem, you must add some lines of code to your .bashrc, .zshenv, or default shell configuration/environment variable file. You can get these lines of code at the authorization page.
